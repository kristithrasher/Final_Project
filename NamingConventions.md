# Branches
`S#_INIT_Description`

__S__: Segment

__#__: Segment Number

__INIT__: Contributor Initials

__Description__: Branch Description


Example: `S1_DG_BanchSample`

# Commits Comments
`S# - INIT - Commit Description`

__S__: Segment

__#__: Segment Number

__INIT__: Contributor Initials

__Description__: Commit Description


Example: `S1 - DG - Commit Description`
